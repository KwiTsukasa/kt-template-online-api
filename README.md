## 技术总览

`Node`、`Ts`、`Nest.js`、`TypeORM`、`MySQL`、`Express`

## 项目简介

此项目为`kt-template-online`服务端,基于`Node`、`Ts`开发

使用服务端框架`Nest.js`构建项目以及`ORM`框架`TypeORM`快速生成`SQL`语句以及映射`SQL`库表字段关系

## 运行项目

```bash
# 运行
$ pnpm start

# 开发环境
$ pnpm start:dev

# 生产环境
$ pnpm start:prod
```

## 测试

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test coverage
$ pnpm test:cov
```
